## ADRSZSW_OneShot.py

実行方法： ./ADRSZSW_OneShot.py
実行すると、スイッチを押している間だけランプが光ります、
また、1秒ごとにスイッチの入力状態とランプの光度(0~255)を表示します。

## ADRSZSW_Trig.py

実行方法： ./ADRSZSW_Trig.py
実行すると、スイッチを押すごとに入力状態とランプが切り替わります。
また、1秒ごとにスイッチの入力状態とランプの光度(0~255)を表示します。
